# Ajmal & Sneha Scroll Frame Invitation

## How to use

1. Put all 100 JPG frames inside the `frames` folder.
2. Frame names must be exactly:
   - `ezgif-frame-001.jpg`
   - `ezgif-frame-002.jpg`
   - ...
   - `ezgif-frame-100.jpg`
3. Upload `index.html`, `style.css`, `script.js`, and the `frames` folder to GitHub.
4. Enable GitHub Pages from repository Settings → Pages.

## Edit wedding details
Open `index.html` and update:
- Date
- Time
- Venue
- Location link
