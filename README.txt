Ajmal & Sneha Scroll Frame Invitation

Add your 100 JPG frames inside the frames folder using this exact naming:
frames/ezgif-frame-001.jpg
...
frames/ezgif-frame-100.jpg

Then upload all files to GitHub Pages root:
index.html
style.css
script.js
frames/

If you change the frame count, update FRAME_COUNT in script.js.
