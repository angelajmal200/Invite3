Ajmal & Sneha Scroll Frame Parallax Website

How to use:
1. Copy all your JPG frames into the frames folder.
2. Frames must be named like:
   ezgif-frame-001.jpg
   ezgif-frame-002.jpg
   ezgif-frame-003.jpg
3. Open script.js.
4. Change FRAME_COUNT to your actual total number of frames.
5. Open index.html or upload the folder to GitHub Pages.

Recommended:
- Use 15 FPS frames.
- Keep total frames below 300 for mobile performance.
- Convert JPG to WebP later for faster loading.
